# TipCalculator - SyncHubCurriculum

This Xcode project is meant to teach the basics of good programming practices. Intended for GT iOS Club Spring 2026.

## ContentView.swift

This file is supposed to be a 'God Object' where it handles the Model, ViewModel, and View portions of the code. There malformed programming practices are marked throughout the file.

## Model, View, ViewModel

These folders are meant to demonstrate the MVVM architecture as well as good programming practices.

### Restaurant.swift

Demonstrates proper use of CamelCase and the model portion of MVVM

### RestaurantViewModel.swift

Demonstrates how to document code, avoid 'Magic Numbers', improve naming convention, and interact with the Model

### RestaurantView.swift

Demonstrates how the business logic of the application should be exclusively in the ViewModel
